# CodeQuest - 编程冒险学院

2026 福建高校"火山杯"Agent 创新大赛参赛作品

## 项目简介

CodeQuest 是一个面向编程学习的 AI 学习中心。项目分为三层：
- 教学环境：关卡、代码提交、JDK 运行、星级评分、解锁进度。
- 学习智能体：代码诊断、苏格拉底式提示、学习画像、路径推荐、总结反馈。
- 展示与激励：商城、皮肤、排行榜、成就系统预留。

## 技术栈

- **前端**：React + TypeScript + Vite + ReactFlow
- **后端**：Node.js + Express
- **数据库**：lowdb + JSON 文件（测试阶段使用 `server/data/db.json`，后续可迁移 MySQL）

## 快速开始

### 安装依赖

```bash
# 安装根目录依赖
npm install

# 安装前后端依赖
cd client && npm install
cd ../server && npm install
```

### 运行项目

```bash
# 在根目录运行（同时启动前后端）
npm run dev
```

或者分别运行：

```bash
# 后端（端口5000）
cd server && npm run dev

# 前端（端口3000）
cd client && npm run dev
```

### 访问

打开浏览器访问 http://localhost:3000

演示账号：用户名 `demo`，密码 `123456`。该账号预置少量通关、提示和学习事件数据，便于直接展示学习画像、成就系统与 AI 辅导闭环。

## 功能说明

### 用户系统
- 注册/登录
- 选择性别
- 角色皮肤定制

### 关卡系统
- 主线规划为100关：Java 入门篇50关 + Java 进阶篇50关
- 当前已开放 Java 入门篇前30关：基础程序结构、变量、运算、字符串、判断、循环、数组、统计、方法、字符串遍历与阶段综合
- Java 入门篇31-50关、Java 进阶篇51-100关已预留为规划关卡，暂时锁定
- 三星评分机制
- 三段式提示：方向提示不降低最高星级，关键概念提示最高2星，局部代码提示最高1星
- 需要通关上一关后才能解锁下一关
- 进度保存

### 学习智能体 MVP
- 代码诊断 Agent：根据编译错误、运行错误和输出偏差生成诊断。
- 苏格拉底式提示 Agent：先给问题引导，再给关键概念，最后才给局部代码提示。
- 学习画像 Agent：记录提交事件和提示事件，统计薄弱知识点与优势知识点。
- 路径推荐 Agent：根据薄弱点推荐复习关、巩固关或下一关。
- 总结反馈 Agent：每关提交后生成掌握点与下一步练习建议。
- 个人中心 Agent 面板：汇总学习风险、关卡进度、3 星掌握率、行动建议和近期学习事件，方便竞赛演示完整学习闭环。

### 算法天梯
- 当前仅预留入口和排行榜展示
- 后续可接入算法题、沙箱判题、时间/空间复杂度评分

### 成就系统
- 通关成就：记录主线关卡阶段性完成
- 掌握成就：记录 3 星完成、连续推进等学习表现
- Agent 成就：记录使用分层提示、持续学习事件等智能体互动
- 成就页展示已解锁数量、分类和解锁时间

### 知识图谱
- 笔记编辑
- 双向链接
- 可视化图谱

### 商城系统
- 星星购买皮肤
- 皮肤装备

### 排行榜
- 班级排名
- 全校排名

### AI辅导
- 关卡页 AI 辅导区
- 三段式提示
- 提交后诊断与总结反馈
